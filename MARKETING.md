## Brand Copy

### One-line Description

Curiohole is a collection of free relaxing browser games you can play instantly with no download or login.

### Short Description

Curiohole is a lightweight browser game site for simple, relaxing games you can play instantly. It focuses on casual puzzle, memory, reaction, and logic games that work directly in the browser with no download, no login, and no app install.

### Long Description

Curiohole is a collection of free relaxing browser games designed for quick breaks and casual play. The site focuses on lightweight games that open fast, work directly in the browser, and require no download, login, or app installation.

Curiohole starts with Ball Sort Puzzle Online and will gradually add more casual puzzle, memory, reaction, logic, and stress-relief games over time. The goal is to make it easy for anyone to open a game and start playing within seconds.

## Ball Sort Puzzle Copy

### One-line Description

Ball Sort Puzzle Online is a free browser puzzle game where you sort colored balls into matching tubes.

### Short Description

Play Ball Sort Puzzle Online for free. Move colored balls between tubes until each tube contains only one color. No download, no login, and no app install required.
